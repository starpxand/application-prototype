## 🏙 О проекте

**GTA VI Web Prototype** — полностью браузерная 3D-игра в жанре «открытый мир», вдохновлённая вселенной Grand Theft Auto VI (сеттинг Miami/Vice). Приложение реализовано в виде **единого HTML-файла** без серверной части и работает прямо в браузере.

## HUD, Frontend, Система управления, представлены на скринах ниже

![Frontend](docs/screenshots/Frontend.png)

![HUD](docs/screenshots/HUD-интерфейс.png)

Проект создан в рамках изучения методологии разработки ПО с применением **ИИ-агентов**: генерацию рабочего кода выполнял агент **Claude (Anthropic)** через 6 итерационных запросов.

### ✨ Реализованные игровые системы

| Система | Описание |
|---------|----------|
| 🏗 **Процедурный город** | Генерируется при каждом запуске: сетка кварталов, здания 3 архитектурных типов (20–100 ед. высоты), деревья, парки, дороги с разметкой |
| 🚶 **ИИ пешеходов (NPC)** | 180 NPC с машиной состояний: `walk` (патруль по waypoints) → `panic` (бегство от игрока, скорость ×3.5) |
| 🚗 **Транспортная система** | 15 автомобилей на дорогах, физика вождения с инерцией, дрифт (ПРОБЕЛ), посадка/высадка (F) |
| 🔫 **Система стрельбы** | Raycasting от центра камеры, автоогонь (100 мс), 30 патронов, перезарядка, эффект отдачи |
| ✨ **Система частиц** | Искры при попаданиях, кровь, гильзы, дымовые эффекты |
| 🖥 **HUD-интерфейс** | Деньги ($), уровень розыска (⭐×5), счётчик патронов, прицел, подсказки взаимодействия |
| 🌆 **Визуальный стиль** | Miami Vice: розово-голубое небо, неоновые акценты (cyan/magenta), процедурные Canvas-текстуры |

## ▶ Как запустить

> **Требования:** любой современный браузер с поддержкой WebGL (Chrome 90+, Firefox 88+, Edge 90+)

```bash
# Открыть html файл в браузере
# Windows:
start game_ai.html
# macOS:
open game_ai.html
# Linux:
xdg-open game_ai.html
```

**Или скачать напрямую:** [`game_ai.html`](game_ai.html) → открыть в браузере.

## 🎮 Управление

### Пешком

| Клавиша | Действие |
|---------|----------|
| `W` `A` `S` `D` | Движение |
| `SHIFT` | Бег (скорость ×2) |
| `ПРОБЕЛ` | Прыжок |
| `F` | Сесть в машину |
| `ЛКМ (удержание)` | Авто-огонь |
| `ESC` | Пауза / освободить курсор |

### За рулём

| Клавиша | Действие |
|---------|----------|
| `W` / `S` | Газ / тормоз |
| `A` / `D` | Руль влево / вправо |
| `ПРОБЕЛ` | Дрифт |
| `F` | Выйти из машины |

## 🤖 Разработка с помощью ИИ-агента

Проект создан с применением **Claude (claude-sonnet-4)** через итерационные запросы. Каждый этап расширял функционал предыдущей версии:

```
Запрос 1 → Базовая сцена + управление WASD (PointerLockControls) + образ оружия
Запрос 2 → Исправление зданий + процедурные текстуры Canvas API (асфальт, фасады)
Запрос 3 → NPC с ИИ-поведением (walk/panic + waypoints) + образы машин
Запрос 4 → Транспорт с физикой вождения и дрифтом + трава + насыщенная цветовая схема
Запрос 5 → Оружие + Raycasting + HUD + деревья + закат + окна + полосы движения
Запрос 6 → Miami Vice-стиль + полицейские мигалки + сбивание NPC + деревья + оптимизация
```
